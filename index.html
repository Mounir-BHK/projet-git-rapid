Qu'est-ce que Git ?

Git est un système de gestion de versions. Il permet de suivre les modifications d’un projet et de travailler efficacement en équipe.

Pourquoi utiliser Git ?

  • Suivre l'historique des modifications
  • Travailler à plusieurs sur le même projet
  • Revenir à une version précédente facilement
  • Sauvegarder son projet sur GitHub

Commandes Git importantes

  • git init : créer un dépôt Git
  • git add : préparer les fichiers à enregistrer
  • git commit : sauvegarder les modifications
  • git push : envoyer le projet vers GitHub
